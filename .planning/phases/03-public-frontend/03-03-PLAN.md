3. Create `src/components/map/MapViewController.tsx` ('use client'):
   - Must be a child of MapContainer (uses `useMap()` hook)
   - Accept props: `province: string | null`, `flyToTarget: { lat: number; lng: number; venueId: number } | null`
   - useEffect watching `province`:
     - If province is a key in PROVINCE_BOUNDS: `map.fitBounds(PROVINCE_BOUNDS[province])`
     - If province is null (cleared): `map.fitBounds(ATLANTIC_CANADA_BOUNDS)` to zoom back out
   - useEffect watching `flyToTarget`:
     - If flyToTarget is set: `map.flyTo([flyToTarget.lat, flyToTarget.lng], 15)` to pan and zoom to the venue
     - After flyTo completes (use map `moveend` event listener), programmatically open the popup for the marker with the matching venueId. To do this, MapClient must pass a `markersRef` (a Map<number, L.Marker>) into MapViewController, or MapViewController can dispatch a custom event that ClusterLayer listens to.
     - Recommended approach: MapClient maintains a `markersRef = useRef<Map<number, L.Marker>>(new Map())` populated by ClusterLayer (each Marker calls `markersRef.current.set(venueId, markerInstance)` via a ref callback prop). MapViewController receives this ref and after flyTo, calls `markersRef.current.get(venueId)?.openPopup()`.
   - This component renders null — it's a pure side-effect hook component
