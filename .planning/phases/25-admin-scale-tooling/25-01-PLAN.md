**Part B: Instrument all cron routes**

For each of the 6 cron route files, add instrumentation that inserts a `discovery_runs` row. Pattern:

```typescript
import { db } from '@/lib/db/client';
import { discovery_runs } from '@/lib/db/schema';

// Inside GET handler:
const startedAt = new Date();
try {
  const result = await runXxxDiscovery(...);
  await db.insert(discovery_runs).values({
    discovery_method: 'METHOD',
    province: 'PROVINCE_OR_NULL',
    started_at: startedAt,
    completed_at: new Date(),
    candidates_found: result.candidatesFound,
    auto_approved: result.autoApproved,
    queued_pending: result.QUEUED_FIELD,
    skipped_dedup: result.DEDUP_FIELD,
    errors: result.errors,
  });
  return Response.json({ success: true, ...result, timestamp: new Date().toISOString() });
} catch (err) {
  // Error-path insert — wrapped in its own try/catch to avoid masking original error
  try {
    await db.insert(discovery_runs).values({
      discovery_method: 'METHOD',
      province: 'PROVINCE_OR_NULL',
      started_at: startedAt,
      completed_at: new Date(),
      candidates_found: 0, auto_approved: 0, queued_pending: 0, skipped_dedup: 0,
      errors: 1,
      error_detail: String(err),
    });
  } catch (insertErr) {
    console.error('Failed to log discovery run error:', insertErr);
  }
  console.error('Discovery failed:', err);
  return Response.json({ success: false, error: String(err) }, { status: 500 });
}
```

Field mapping per channel:
- **Places routes (x4):** discovery_method='google_places', province='NS'/'NB'/'PEI'/'NL', candidatesFound=result.candidatesFound, autoApproved=result.autoApproved, queuedPending=result.stagedPending, skippedDedup=result.enriched, errors=result.errors
- **Reddit route:** discovery_method='reddit_gemini', province=null, candidatesFound=result.candidatesFound, autoApproved=result.autoApproved, queuedPending=result.staged, skippedDedup=0, errors=result.errors
- **Gemini route:** discovery_method='gemini_google_search', province=null, candidatesFound=result.candidatesFound, autoApproved=result.autoApproved, queuedPending=result.queuedPending, skippedDedup=0, errors=result.errors

IMPORTANT: The error-path `db.insert` must be wrapped in its own try/catch that logs but does not rethrow — see Pitfall 4 in RESEARCH.md.
