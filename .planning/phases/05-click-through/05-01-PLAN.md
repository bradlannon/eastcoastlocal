<feature>
  <name>findNearbyVenues spatial proximity function</name>
  <files>src/lib/timelapse-utils.ts, src/lib/timelapse-utils.test.ts</files>
  <behavior>
    findNearbyVenues takes a click coordinate {lat, lng}, a Map of venue groups (venueId -> {venue, events[]}), and a radius in meters. It returns an array of matching venue groups.

    Distance calculation: Use Haversine formula (pure math, no Leaflet dependency). The formula: `a = sin2(dlat/2) + cos(lat1)*cos(lat2)*sin2(dlng/2); c = 2*atan2(sqrt(a), sqrt(1-a)); d = R*c` where R = 6371000 meters.

    Cases:
    - Single venue at Halifax (44.6488, -63.5752), click at same location -> returns [venue] (distance ~0m, within 2000m)
    - Single venue at Halifax, click at Moncton (46.0878, -64.7782) -> returns [] (distance ~180km, outside 2000m)
    - Two venues both within 500m of click -> returns [venue1, venue2]
    - Venue with null lat -> skipped, not in results
    - Empty venue map -> returns []
    - Custom radius: venue at 1500m, radius=1000 -> returns []; radius=2000 -> returns [venue]

    Also export: `CLICK_RADIUS_METERS = 2000` constant.
    Also export: `VenueGroup` type: `{ venue: Venue; events: EventWithVenue[] }`.
  </behavior>
  <implementation>
    Add to the bottom of `src/lib/timelapse-utils.ts`:

    1. Export `CLICK_RADIUS_METERS = 2000` constant
    2. Export `VenueGroup` interface: `{ venue: Venue; events: EventWithVenue[] }`
    3. Export a pure `haversineDistance(lat1, lng1, lat2, lng2): number` function returning meters (no Leaflet dependency — enables Node testing)
    4. Export `findNearbyVenues(clickLat: number, clickLng: number, venueGroups: Map<number, VenueGroup>, radiusMeters?: number): VenueGroup[]`
       - Default radius = CLICK_RADIUS_METERS
       - Iterate venueGroups, skip entries where venue.lat or venue.lng is null
       - Use haversineDistance to compute distance
       - Return groups within radius

    Import `Venue` type from `@/types/index` (already imported as `EventWithVenue` includes it).

    Add tests to the existing `src/lib/timelapse-utils.test.ts` file in a new describe block at the bottom.
  </implementation>
