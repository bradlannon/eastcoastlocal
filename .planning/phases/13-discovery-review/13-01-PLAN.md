Create the server component page at src/app/admin/discovery/page.tsx:

2. Accept searchParams with a `status` query param (default: 'pending'). Valid values: 'pending', 'approved', 'rejected'.
3. Run two parallel DB queries via Promise.all:
   - Fetch candidates: `db.select().from(discovered_sources).where(eq(discovered_sources.status, status)).orderBy(desc(discovered_sources.discovered_at))`
   - Fetch counts per status: three count queries (one per status) using `db.select({ count: count() }).from(discovered_sources).where(eq(discovered_sources.status, statusValue))` — returns `{ pending: number, approved: number, rejected: number }`
4. Pass candidates array and counts object as props to `<DiscoveryList>` client component.
5. Also pass `activeStatus` string prop so the client knows which tab is active.
