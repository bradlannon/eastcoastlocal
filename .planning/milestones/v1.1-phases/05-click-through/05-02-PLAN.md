      - `<VenuePopup>` for that venue's events
    - Add `max-h-[300px] overflow-y-auto` on the outer container for multi-venue to prevent oversized popups

    **HeatmapClickLayer.tsx** -- Map-level click handler + proximity + Popup:
    - Props interface:
      ```
      interface HeatmapClickLayerProps {
        allEvents: EventWithVenue[];
        timePosition: number;
        referenceDate: Date;
        onPause: () => void;
      }
      ```
    - Import `useMap` from `react-leaflet`, `Popup` from `react-leaflet`
    - Import `findNearbyVenues`, `filterByTimeWindow`, `positionToTimestamp`, `computeVenueHeatPoints`, `VenueGroup`, `CLICK_RADIUS_METERS` from `@/lib/timelapse-utils`
    - Import `HeatmapPopup` from `./HeatmapPopup`
    - Import `L` from `leaflet` (for `L.LatLng` type)

    Internal state: `clickState: { latlng: L.LatLng; venues: VenueGroup[] } | null`

    useMemo to derive `venueGroups: Map<number, VenueGroup>` from time-windowed events:
    1. `const center = positionToTimestamp(timePosition, referenceDate)`
    2. `const windowed = filterByTimeWindow(allEvents, center.getTime(), 24)`
    3. Group windowed events by `venues.id` into a `Map<number, VenueGroup>` (same pattern as computeVenueHeatPoints but keeping full venue+events)

    useEffect for map click handler:
    1. `map.on('click', handler)` -- return cleanup `map.off('click', handler)`
    2. Handler: call `findNearbyVenues(e.latlng.lat, e.latlng.lng, venueGroups)`
    3. If results empty, return (do nothing -- allows normal map click behavior)
    4. Call `onPause()` FIRST (per research Pitfall 4 -- pause before setting state)
    5. Then `setClickState({ latlng: e.latlng, venues: results })`
    6. Dependencies: `[map, venueGroups, onPause]`

    useEffect cleanup on unmount: set `clickState` to null (per research Pitfall 2 -- popup cleanup on mode switch)

    Render: return a fragment. If `clickState` is not null, render:
    ```
    <Popup position={clickState.latlng} onClose={() => setClickState(null)}>
      <HeatmapPopup venues={clickState.venues} />
    </Popup>
    ```
    Use `<Popup position={latlng}>` pattern (NOT `<Marker><Popup>` -- per research Pitfall 1).

    IMPORTANT: Clear clickState when timePosition changes (add useEffect that sets clickState to null when timePosition changes). This prevents stale popups showing wrong-window events after scrubbing.
