   a. Module-level `domainLastRequest = new Map<string, number>()` and `DOMAIN_MIN_GAP_MS = 2000`.

   b. `applyDomainRateLimit(url)` — checks elapsed time since last request to same domain, delays if under gap + random 0-500ms jitter. Updates map BEFORE the request.

   d. `fetchPage(url)` — internal helper: calls `applyDomainRateLimit(url)`, then `fetchWithRetry(url)`. Returns `{ html: string, text: string }` where html is raw response.text(), and text is cheerio-cleaned content (same stripping logic as current code).

   e. `detectNextPageUrl(html, baseUrl)` — uses cheerio selectors: `a[rel="next"], a[aria-label*="next" i], a.pagination-next, li.next > a`. Returns absolute URL via `new URL(href, baseUrl)` or null.

   f. Updated `fetchAndPreprocess(url, options?: { maxPages?: number })` signature returning `Promise<{ text: string; rawHtml: string }>`:
      - Hard cap: `Math.min(options?.maxPages ?? 1, 3)`
      - Loop: fetch pages, accumulate text, capture rawHtml from first page BEFORE cheerio mutation
      - 500ms delay between pages within same source
      - Apply `slice(0, 15_000)` to accumulated text at the end (not per-page)
      - CRITICAL: Capture rawHtml = html BEFORE cheerio.load() strips script tags
