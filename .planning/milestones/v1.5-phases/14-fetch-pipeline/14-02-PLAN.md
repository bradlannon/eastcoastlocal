Modify `src/lib/scraper/orchestrator.ts` with these changes:

1. Add import for `extractJsonLdEvents` from `./json-ld`.

2. Add `HTTP_THROTTLE_MS` constant:
   ```typescript
   const HTTP_THROTTLE_MS = parseInt(process.env.HTTP_THROTTLE_MS ?? '1000', 10);
   ```

3. In the `venue_website` branch of the source loop, replace:
   ```typescript
   const pageText = await fetchAndPreprocess(source.url);
   const extracted = await extractEvents(pageText, source.url);
   ```
   With:
   ```typescript
   const { text, rawHtml } = await fetchAndPreprocess(source.url, {
     maxPages: source.max_pages ?? 1,
   });

   // JSON-LD fast path: structured data is authoritative (confidence=1.0)
   // Short-circuit — do NOT also call Gemini if JSON-LD found events
   const jsonLdEvents = extractJsonLdEvents(rawHtml);

   let extracted: ExtractedEvent[];
   if (jsonLdEvents.length > 0) {
     extracted = jsonLdEvents;
     console.log(`  ✓ ${venue.name}: ${extracted.length} events (JSON-LD, skipping Gemini)`);
   } else {
     extracted = await extractEvents(text, source.url);
     console.log(`  ✓ ${venue.name}: ${extracted.length} events`);
   }

   for (const event of extracted) {
     await upsertEvent(source.venue_id, event, source.url);
   }
   ```

4. Add the `ExtractedEvent` type import from `@/lib/schemas/extracted-event`.

5. Move the existing AI_THROTTLE_MS delay to only apply when Gemini was actually called (inside the `else` branch or after the if/else). The AI throttle should NOT apply after JSON-LD extraction since no Gemini call was made.

6. Add inter-source HTTP throttle AFTER source completion (before moving to next source in the loop), applying only to venue_website sources:
   ```typescript
   // HTTP throttle between venue_website sources (separate from AI throttle)
   if (HTTP_THROTTLE_MS > 0) {
     await delay(HTTP_THROTTLE_MS);
   }
   ```
   This goes after the upsert loop, before the `successCount++` or at end of the venue_website branch.

7. Remove the old `console.log` for venue success (replaced by the one inside the if/else above).

ANTI-PATTERN: Do NOT call both extractJsonLdEvents AND extractEvents for the same source. JSON-LD short-circuits Gemini entirely.
ANTI-PATTERN: Do NOT apply AI_THROTTLE_MS after JSON-LD extraction — no Gemini call was made, so no AI rate limit needed.
