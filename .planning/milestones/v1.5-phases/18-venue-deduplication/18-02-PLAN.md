    1. Load city venues: `db.query.venues.findMany({ where: ilike(venues.city, city), columns: { id: true, name: true, lat: true, lng: true } })` — use `ilike` not `eq` for city to handle case mismatches (Pitfall 2 from research).
    2. For each city venue, call `scoreVenueCandidate({ name, lat: null, lng: null }, candidate)` — TM venues have no lat/lng at creation time (Key Discovery from research).
    3. Call `findBestCandidate(decisions)` to get the best result.
    4. If best is `merge`:
       - If NOT dryRun: check `scrape_sources` for duplicate venue (Pitfall 3). If scrape_sources exist, log warning and skip delete — just reassign events and return canonical ID. If no scrape_sources, reassign events then delete duplicate venue row.
       - IMPORTANT: Since TM venues have no geo, the merge action will only fire on re-ingest (when the TM venue was previously created and geocoded). On first ingest, incoming geo is null, so scoring returns 'review' not 'merge'.
       - Return `canonicalId`.
    5. If best is `review`: insert a row into `venue_merge_candidates` table with all scores/reason. Continue to insert new venue.
    6. If null (all keep_separate): insert new venue as before.
    7. If dryRun: log decisions to console but make NO DB writes (no inserts, no deletes, no merge candidate rows).

    Import from venue-dedup.ts: `scoreVenueCandidate`, `findBestCandidate`, `DedupeDecision`.
    Import from schema.ts: `venue_merge_candidates`.
    Import `ilike` from drizzle-orm (already imported).

    Also add a `mergeVenueDuplicate` helper function:
    ```typescript
    async function mergeVenueDuplicate(duplicateId: number, canonicalId: number): Promise<void> {
      // Check for scrape_sources referencing the duplicate
      const sources = await db.query.scrape_sources.findMany({
        where: eq(scrape_sources.venue_id, duplicateId),
      });
      if (sources.length > 0) {
        console.warn(`[venue-dedup] Duplicate venue ${duplicateId} has ${sources.length} scrape sources — reassigning events only, not deleting`);
      }
      // Reassign all events from duplicate to canonical
      await db.update(events).set({ venue_id: canonicalId }).where(eq(events.venue_id, duplicateId));
      // Delete only if no scrape_sources
      if (sources.length === 0) {
        await db.delete(venues).where(eq(venues.id, duplicateId));
      }
    }
    ```
