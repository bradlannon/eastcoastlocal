  <behavior>
    Eventbrite:
    - scrapeEventbrite parses org ID from source.url format "eventbrite:org:12345678"
    - scrapeEventbrite calls GET /v3/organizations/{org_id}/events/?status=live&expand=venue with Bearer token
    - scrapeEventbrite skips events with start date in the past
    - scrapeEventbrite maps Eventbrite event fields to ExtractedEvent shape (performer=name.text, event_date=start.utc sliced, event_time=start.local sliced, confidence=1.0)
    - scrapeEventbrite calls upsertEvent for each valid event
    - scrapeEventbrite throws on non-OK HTTP response

    Bandsintown:
    - scrapeBandsintown parses artist name from source.url format "bandsintown:artist:The+Trews"
    - scrapeBandsintown calls GET /artists/{name}/events/?app_id={id}&date=upcoming
    - scrapeBandsintown filters events to Atlantic Canada only (NB, NS, PE/PEI, NL provinces)
    - scrapeBandsintown skips past events
    - scrapeBandsintown maps Bandsintown event fields to ExtractedEvent shape (confidence=1.0)
    - scrapeBandsintown calls upsertEvent for each valid event
    - scrapeBandsintown throws on non-OK HTTP response
  </behavior>
  <action>
    1. Create `src/lib/scraper/eventbrite.ts`:
       - Export `scrapeEventbrite(source: ScrapeSource): Promise<void>`
       - Parse org ID: `source.url.replace('eventbrite:org:', '')`
       - Fetch `https://www.eventbriteapi.com/v3/organizations/${orgId}/events/?status=live&expand=venue` with `Authorization: Bearer ${process.env.EVENTBRITE_TOKEN}`
       - Throw on non-OK response
       - Loop events: skip if `new Date(event.start.utc) < new Date()`, then call `upsertEvent(source.venue_id, { performer: event.name.text, event_date: event.start.utc.slice(0, 10), event_time: event.start.local.slice(11, 16), price: null, ticket_link: event.url, description: event.description?.text?.slice(0, 500) ?? null, cover_image_url: event.logo?.url ?? null, confidence: 1.0 }, event.url)`
       - Import upsertEvent from `./normalizer`

    2. Create `src/lib/scraper/bandsintown.ts`:
       - Export `scrapeBandsintown(source: ScrapeSource): Promise<void>`
       - Parse artist name: `source.url.replace('bandsintown:artist:', '')`
       - Fetch `https://rest.bandsintown.com/artists/${encodeURIComponent(artistName)}/events/?app_id=${process.env.BANDSINTOWN_APP_ID}&date=upcoming`
       - Throw on non-OK response
       - Loop events: filter to Atlantic Canada by checking `venue.region` against ['New Brunswick', 'Nova Scotia', 'Prince Edward Island', 'Newfoundland and Labrador', 'NB', 'NS', 'PE', 'NL']. Skip if not Atlantic Canada. Skip if `new Date(event.datetime) < new Date()`.
       - Call `upsertEvent(source.venue_id, { performer: artistName.replace(/\+/g, ' '), event_date: event.datetime.slice(0, 10), event_time: event.datetime.slice(11, 16), price: null, ticket_link: event.offers?.[0]?.url ?? event.url, description: event.description ?? null, cover_image_url: null, confidence: 1.0 }, event.url)`

Verify: only Atlantic Canada events pass filter, past events skipped, upsertEvent called with correct fields, HTTP error throws. Verify app_id included in URL.
