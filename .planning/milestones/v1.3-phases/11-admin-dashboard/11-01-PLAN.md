**Dashboard page (src/app/admin/page.tsx):**
- Server component with `export const dynamic = 'force-dynamic'`
- Import db from '@/lib/db/client' and tables from '@/lib/db/schema'
- Import { count, eq, desc, max, sql } from 'drizzle-orm' as needed

**Data queries (run in parallel with Promise.all):**
1. Total venues: `db.select({ count: count() }).from(venues)`
2. Active sources: `db.select({ count: count() }).from(scrape_sources).where(eq(scrape_sources.enabled, true))`
3. Pending discoveries: `db.select({ count: count() }).from(discovered_sources).where(eq(discovered_sources.status, 'pending'))`
4. Last scrape time: `db.select({ latest: max(scrape_sources.last_scraped_at) }).from(scrape_sources)`
5. Source health rows: `db.select({ sourceId: scrape_sources.id, url: scrape_sources.url, status: scrape_sources.last_scrape_status, lastScrapedAt: scrape_sources.last_scraped_at, enabled: scrape_sources.enabled, venueName: venues.name }).from(scrape_sources).innerJoin(venues, eq(scrape_sources.venue_id, venues.id)).orderBy(sql`CASE WHEN ${scrape_sources.last_scrape_status} = 'failure' THEN 0 WHEN ${scrape_sources.last_scrape_status} = 'pending' THEN 1 ELSE 2 END`)`

**Stat cards row (4 cards in horizontal grid):**
- Use Tailwind grid: `grid grid-cols-1 sm:grid-cols-2 lg:grid-cols-4 gap-4`
- Each card: white bg, rounded-lg, shadow-sm, border, p-6
- Card content: label (text-sm text-gray-500), value (text-3xl font-bold text-gray-900)
- Total Venues card: value = venue count, wrapped in Link to /admin/venues
- Active Sources card: value = active source count
- Pending Discoveries card: value = pending count, wrapped in Link to /admin/discovery
- Last Scrape card: value = relative time string (e.g., "2 hours ago"), amber text (text-amber-600) if >24h ago

**Relative time helper:** Create a simple function at top of file (not a separate module). Takes a Date|null, returns string like "2 hours ago", "3 days ago", "Never". Use simple math: diff in ms, convert to minutes/hours/days. Return "Never" if null.

**Page heading area:**
- h1 "Admin Dashboard" (text-2xl font-bold)
- Small "Data as of [current time]" text (text-xs text-gray-400) next to heading
- RefreshButton component next to the "Data as of" text

**Source Health table (below stat cards):**
- h2 "Source Health" heading with top margin
- HTML table with Tailwind: w-full, divide-y, bg-white rounded-lg shadow-sm border
- Columns: Venue | Source URL | Status | Last Scraped
- Venue: venue name text
- Source URL: truncated with max-w-xs truncate class, full URL in title attribute for hover
- Status: badge span — green (bg-green-100 text-green-800) for "success", red (bg-red-100 text-red-800) for "failure", gray (bg-gray-100 text-gray-800) for "pending"
- Last Scraped: relative time string using same helper
- Disabled sources: entire row gets opacity-50 for muted appearance
- Empty state: "No scrape sources configured" message if no rows

**Error handling:** Wrap all queries in try/catch. On error, render a simple error message "Failed to load dashboard data" with red styling.
